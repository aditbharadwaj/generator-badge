# Markup for inserting badges

## [Markdown](https://daringfireball.net/projects/markdown/)

```markdown
<!-- badge -->
## Badge will go in here ##
<!-- endbadge -->
```

## [html](https://www.w3.org/html)

```html
<!-- badge -->
## Badge will go in here ##
<!-- endbadge -->
```

## [AsciiDoc](http://www.methods.co.nz/asciidoc/)

```asciidoc
//// badge
## Badge will go in here ##
////
```

## [Textile](https://txstyle.org)

```textile
###. badge
## Badge will go in here ##
###. endbadge
```
